Movie Maniacs

Objective:

Movie Maniacs is an app that connects movie lovers to their favorite movies and fellow cinema enthusiasts. This is a project where I build the server-side API of a cinema themed web app. This web application will provide users with info about different movies, directors, and genres. Users will be able to sign up for an account, update their personal information, and create a list of their favorite movies.

Endpoint Documentation

Request URL HTTP Method Response
Return a list of ALL movies to the user /movies GET A JSON object containing a list of ALL movies
Return data (description, genre, director, image URL) about a single movie by title to the user /movies/title GET A JSON object containing data about a single movie
Return data about a genre by title /movies/genre/genreName GET A JSON object containing data about a specific genre
Return data about a director (bio, birth year, etc.) by name /movies/directors/directorName GET A JSON object containing data about a specific director
Allow new users to create a profile /users POST A JSON object containing data about the new user
Allow users to update their profile /users/id PUT A JSON object containing the updated user information
Allows users to add a movie to their favourites /users/id/movieTitle POST A text message saying the movie has successfully been added to the users favorite movies list
Allows users to remove a movie from their favourites /users/id/movieTitle DELETE A text message saying the movie has successfully been removed from the favorite movies list
Allows existing users to delete their profile /users/id DELETE A text message saying that the user has successfully deleted their profile
